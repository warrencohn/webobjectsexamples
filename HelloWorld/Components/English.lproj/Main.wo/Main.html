This application supports English, French, Spanish, German, Japanese, and Chinese. Change your browser's languages preferences to view the different languages.

This example application demonstrates the communication of between components - specifically how to utilize key-value coding to pass information between objects without having to implement accessor methods.

In this example, we use a form with a single text field and submit button, and a single method in the Main component to pass your name from one page to the next.
